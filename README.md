# Amrutha Gowri Portfolio Website

This is a professional multi-page GitHub Pages portfolio website for Amrutha Gowri Jayasimha Hanumesh.

## Pages

- Home: `index.html`
- Projects: `projects.html`
- Research: `research.html`
- Experience: `experience.html`
- Contact: `contact.html`

## How to Use

1. Create a GitHub repository named `AmruthaGowri.github.io`.
2. Upload all files from this folder.
3. Go to repository Settings → Pages.
4. Select Branch: `main`, Folder: `/root`.
5. Save and open `https://AmruthaGowri.github.io`.
