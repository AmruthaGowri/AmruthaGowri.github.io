Featured Project

Multi-Trigger Backdoor Attacks in Code-Generating LLMs

This project studies how code-generating large language models can be poisoned using hidden triggers during LoRA fine-tuning. I designed a multi-trigger backdoor poisoning framework using CodeGen-350M, LoRA adapters, PyTorch, and HuggingFace Transformers.
